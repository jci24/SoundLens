# SoundLens Frontend

React, TypeScript, Vite, Tailwind CSS, and shadcn/ui with Radix primitives.

This app is currently a scaffold only. Product implementation starts in later vertical slices.

## Commands

```bash
npm install
npm run lint
npm run build
npm run dev
```

## Conventions

- Use semantic design tokens from `src/index.css`.
- Add shadcn/ui components intentionally, not wholesale.
- Keep acoustic visualizations evidence-focused and inspectable.
- Do not add feature folders until a real vertical slice needs them.
